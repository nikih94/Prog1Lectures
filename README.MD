## Programming 1 – Lecture Code

This repository contains the code written during Programming 1 lectures.
It includes material from both the English (EN) and Slovenian (SLO) groups.

### Structure

All source files are located in the src folder, organized as follows:

```
src/
├── EN/   # Code from the English group lectures
│    ├── P1/   # Lecture 1
│    ├── P2/   # Lecture 2
│    └── ...
└── SLO/  # Code from the Slovenian group lectures
├── P1/   # Lecture 1
├── P2/   # Lecture 2
└── ...
```


>Each Px subpackage corresponds to a specific lecture (e.g., P1 = first lecture, P2 = second lecture, and so on).
